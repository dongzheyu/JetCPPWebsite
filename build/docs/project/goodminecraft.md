## 📥 **安装指南**

### 🛠️ **快速安装**
1. **下载整合包** 📦
   - 确保你有最新的 Fabric 加载器
   - 下载 GoodM 整合包文件

2. **安装步骤** 🔧
   ```
   1. 备份你的 .minecraft 文件夹
   2. 将整合包文件解压到 .minecraft 文件夹
   3. 启动游戏，选择 Fabric 配置档
   4. Enjoy! 🎉
   ```

3. **首次启动** ✨
   - 等待模组加载完成
   - 根据需要调整光影设置
   - 配置小地图快捷键
   - 设置投影快捷键

## 更新日志 v2.0.0

### 🎨 新增材质包

本次更新添加了6个精心挑选的材质包，全面提升游戏视觉体验：

#### 1. §bTorturable Healthbars
- **功能**: 可折磨血条显示改进
- **特点**: 优化生物血条显示效果，提供更清晰的生命值指示

#### 2. Angel's Weather v1.9.1.1hf
- **功能**: 天使天气系统
- **特点**: 增强天气效果，提供更逼真的气象变化体验

#### 3. Blue's Better Monsters v0.12
- **功能**: 蓝色优化怪物
- **特点**: 改进怪物外观和细节，使敌对生物更加生动

#### 4. Nature X - 14.10
- **功能**: 自然X材质包
- **版本**: 兼容 Minecraft 1.21.8-1.21.10
- **特点**: 全面优化自然环境纹理，提升草地、树木、水体等视觉效果

#### 5. rainbows-foliage-polytone-edition v1.2
- **功能**: 彩虹多色调植被版
- **特点**: 为游戏中的植被添加多彩效果，营造梦幻般的自然景观

#### 6. Visual Effects+
- **功能**: 视觉效果增强
- **特点**: 强化游戏中的各种视觉效果，提升整体视觉冲击力

### ⚙️ 兼容性说明
- **Minecraft版本**: 主要兼容 1.21.8-1.21.10
- **性能影响**: 中等，建议配置较好的设备使用
- **安装方式**: 将所有材质包文件放入游戏资源包文件夹即可

### 🔄 安装步骤
1. 解压所有材质包文件
2. 打开 Minecraft 游戏
3. 进入选项 → 资源包
4. 将解压后的材质包文件夹拖入资源包列表
5. 确认启用所有材质包
6. 重启游戏生效

### 📝 注意事项
- 请确保游戏版本与材质包兼容
- 建议按顺序启用材质包以获得最佳效果
- 如遇到性能问题，可适当减少启用的材质包数量
